Example 1: If-Else Conditional

int x = 5;
int y = 10;
if (x > y) {
    x = x + y;
} else {
    y = y - x;
}

What are the final values of x and y?

Example 2: For Loop

int sum = 0;
for (int i = 1; i <= 5; i++) {
    sum += i;
}

What is the final value of sum?

Example 3: While Loop

int a = 5;
while (a > 0) {
    a--;
}

How many iterations does the loop run?

Example 4: Nested Loops

int count = 0;
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        count++;
    }
}

What is the final value of count?

Example 5: Switch Case

int num = 2;
switch (num) {
    case 1:
        num += 5;
        break;
    case 2:
        num += 10;
        break;
    default:
        num += 0;
}

What is the final value of num?

Example 6: Compound Conditional

int a = 10, b = 20;
if (a > 5 && b < 25) {
    a = b;
}

What is the final value of a?

Example 7: Do-While Loop

int i = 0;
do {
    i += 2;
} while (i < 10);

What is the final value of i?

Example 8: Combination of For Loop and If Statement

for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        printf("%d is even\n", i);
    } else {
        printf("%d is odd\n", i);
    }
}

How many times is "even" printed?

Example 9: Nested If-Else

int x = 15;
if (x > 10) {
    if (x % 2 == 0) {
        x = x / 2;
    } else {
        x = x * 2;
    }
} else {
    x = 0;
}

What is the final value of x?

Example 10: For Loop with Break Statement

int sum = 0;
for (int i = 1; i <= 10; i++) {
    if (sum > 20) break;
    sum += i;
}

At which value of i does the loop terminate?
